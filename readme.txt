# Delivery Service

Это приложение предназначено для фильтрации заказов по району и времени доставки. 
Оно использует принципы чистой архитектуры и организовано по слоям.


## Настройка

1. В DeliveryService.ConsoleApp/Data в файле orders.csv укажите данные для фильтрации 
(файл должен быть в формате .csv)

2. В файле конфигурации appsettings.json в DeliveryService.ConsoleApp указываются пути 
для определения того где программе брать список заказовб, где сохранять отфильтрованные заказы и где сохранять логи операций 
по желанию пути можно изменить.

## Запуск 

Чтобы запустить приложение, выполните следующую команду из папки DeliveryService.ConsoleApp:

dotnet run <district> <firstDeliveryTime>

Где:
   <district> — район доставки (например, Downtown).
   <firstDeliveryTime> — время первой доставки в формате гггг-ММ-дд ЧЧ:мм:сс (например, 2024-10-24 12:00:00).

### Пример запуска

dotnet run Downtown "2024-10-24 12:00:00"

### Результат

После выполнения программы:
   1. Отфильтрованные заказы будут сохранены в файл Data/filtered_orders.csv.
   2. Логи будут записаны в файл logs/delivery_log.txt.
